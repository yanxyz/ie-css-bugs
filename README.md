# IE CSS Bugs

点击查看 [IE CSS Bugs](https://yanxyz.github.io/css-ie-bugs)

有错误或补充，欢迎提交 issue 。

## 怎么做

在 `bugs.md` 中添加讲解，将测试文件在 `demo` 目录。

安装 [pandoc](http://johnmacfarlane.net/pandoc/) 及 node > 0.11

命令行运行 `node build.js`，生成 `index.html`。
